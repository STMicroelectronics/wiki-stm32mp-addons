#############
A35-TD flavor
#############

With A35-TD flavor, the  Arm® Cortex®-A is the owner of Trusted Domain (TDCID), is in charge of:

* firmware: authentification, loading, remote control (start/stop)
* clock tree setting, except clock gating of hardware blocks
* security setup (memories, peripherals)

This document contains specific topic for M33 copro in stm32mp2 platform.
