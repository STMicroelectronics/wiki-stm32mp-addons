M33-TD flavor
=============

 With A35-TD flavor, the  Arm® Cortex®-M is the owner of Trusted Domain (TDCID), is in charge of:

* security setup (memories, peripherals) for all platform (Arm® Cortex®-A, Arm® Cortex®-M)
* clock tree setting.
* start Arm® Cortex®-A.

This document contains specific topic for M33-TD flavor in stm32mp2 platform.
